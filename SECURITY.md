# Security Policy

## Supported Versions

Seule la branche `main` est supportée.

## Reporting a Vulnerability

Merci d’éviter les issues publiques pour des vulnérabilités.

Procédure recommandée :

1) Préparer un rapport avec : description, impact, étapes de repro, version/commit.
2) Contacter le mainteneur via un canal privé (email/DM) et fournir le rapport.

Si tu n’as pas de canal privé, ouvre une issue **sans détails exploitables** et demande un moyen de contact.
