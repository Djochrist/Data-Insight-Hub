# Contribuer

Merci de vouloir contribuer.

## Pré-requis

- Node.js 20+

## Setup

```bash
npm install
cp .env.example .env
```

## Qualité

Avant d’ouvrir une PR :

```bash
npm run check
npm run build:vercel
```

## Style

- Préférer des changements petits et ciblés.
- Éviter d’ajouter des dépendances non nécessaires.
- Documenter les changements fonctionnels dans `README.md` si besoin.
